# Assets Structure

Place brand files here and keep file names stable to avoid changing HTML.

## Folders
- `logo/`: company logo files.
- `icons/`: floating action icons and social icons.
- `images/hero/`: hero banners.
- `images/gallery/`: intro gallery images.
- `images/products/`: product card images.
- `images/showroom/`: showroom/map visuals.

## Current placeholders
- `logo/the-anh-jp-logo.svg`
- `icons/call.svg`
- `icons/mail.svg`
- `icons/zalo.svg`

You can replace each file with your own icon/image and keep the same file name.
